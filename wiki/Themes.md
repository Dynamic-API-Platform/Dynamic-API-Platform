# UI Themes

Full guide: [UI Themes](https://dynamic-api-platform.github.io/Dynamic-API-Platform/themes/)

Four themes (v1.5.9+): Dark, Light, Ocean, Forest. Header **Live** badge: [Live UI](Live-UI).

## Themes

| Theme | Description |
|-------|-------------|
| **Dark** | Slate + cyan (default) |
| **Light** | Bright daytime UI |
| **Ocean** | Deep navy + teal — calm, technical |
| **Forest** | Charcoal green + emerald — easy on the eyes |

## Switch

- **Header** — palette icon cycles themes
- **Login** — link below sign-in
- **Settings → Display** — default theme for new users

Choice is saved in `localStorage` per browser.
