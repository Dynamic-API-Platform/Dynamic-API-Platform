See [CONTRIBUTING.md](https://github.com/Dynamic-API-Platform/Dynamic-API-Platform/blob/main/CONTRIBUTING.md)

1. Fork the repo
2. Create feature branch (`cursor/short-description` or your convention)
3. Make changes
4. Run tests and build
5. Open Pull Request

```bash
cd backend && npm test && npm run build
cd ../frontend && npm run build
```

Optional: `npm run test:load` with backend running — see [Testing](Testing).
